# cases/ — your private case files live here

Every bill you work on gets its own folder in this directory:

```
cases/
  2026-07-st-marys-er/
    case.md          ← the case file: facts, timeline, strategy, next steps
    bill-original.pdf
    bill-itemized.pdf
    letters/         ← letters generated for this case
    call-log.md      ← notes from phone calls
```

**Everything in this directory is gitignored.** Your medical bills,
diagnoses, account numbers, and personal details stay on your machine.
Never commit them, never paste them into issues or pull requests.

If you contribute improvements back to this project, share the *pattern*
that worked (tactic, template tweak, objection + comeback) with all
personal details removed — not your case file.
